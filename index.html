Gerações dos Computadores

(Voltar)

  1. Primeira Geração (1951-1959): Os computadores da primeira geração usavam válvulas eletrônicas, que eram grandes, consumiam muita energia e produziam muito calor. Eram programados em linguagem de máquina e usavam cartões perfurados para entrada de dados. Exemplos famosos incluem o ENIAC e o UNIVAC.

  2. Segunda Geração (1959-1965): Com a introdução dos transistores, os computadores ficaram menores, mais rápidos, mais eficientes e mais confiáveis. Essa geração usava linguagem de programação de alto nível como COBOL e FORTRAN. Os computadores começaram a ser usados comercialmente em empresas.

  3. Terceira Geração (1965-1971): Os circuitos integrados substituíram os transistores, levando à criação de computadores ainda menores e mais potentes. Essa geração introduziu o conceito de sistemas operacionais e a capacidade de executar várias tarefas simultaneamente (multiprogramação).

  4. Quarta Geração (1971-até os dias atuais): O desenvolvimento do microprocessador marcou o início dessa geração, tornando os computadores pessoais (PCs) uma realidade. Os computadores se tornaram mais acessíveis, potentes e integrados, com avanços na interface gráfica, redes e internet.

  5. Quinta Geração: Ainda em desenvolvimento, essa geração foca na inteligência artificial, aprendizado de máquina, computação quântica e processamento paralelo. A ideia é criar máquinas que possam processar informações de forma semelhante ao cérebro humano, com capacidades de aprendizado e tomada de decisões autônomas.

Surgimento da Internet

(Voltar)
  • A internet surgiu a partir de um projeto militar dos Estados Unidos durante a Guerra Fria, chamado ARPANET, desenvolvido pela Advanced Research Projects Agency (ARPA) no final da década de 1960. O objetivo inicial era criar uma rede de comunicação descentralizada que pudesse resistir a ataques e garantir a troca de informações entre instituições militares e acadêmicas.
  • 1969: A ARPANET conectou pela primeira vez quatro universidades norte-americanas, estabelecendo a primeira rede de computadores do mundo.
  • Anos 1970: Surgiram os primeiros protocolos de comunicação, como o TCP/IP, que permitiram a integração de redes distintas.
  • Anos 1980: A internet começou a se expandir para além do uso militar e acadêmico.
  • 1991: Com a criação da World Wide Web por Tim Berners-Lee, a internet se tornou acessível ao público em geral.

Estrutura Básica do HTML

(Voltar)
  • estrutura básica do HTML (HyperText Markup Language) é composta por elementos que formam a base de qualquer página web. Aqui está um resumo dessa estrutura:
  • < html>: É a raiz do documento HTML. Todos os outros elementos estão contidos dentro dessa tag, definindo o início e o fim da página.
  • < head>: Contém metadados sobre o documento, como o título da página (< title>), links para arquivos CSS, configurações de caracteres (< meta charset="UTF-8">), scripts JavaScript e outras informações de configuração.
  • < body>: Contém todo o conteúdo visível da página, como textos, imagens, links, tabelas, formulários e outros elementos que compõem a interface com o usuário.
  • < title>: Define o título da página que aparece na aba do navegador e é importante para SEO

Tags

(Voltar)

As tags HTML são os blocos de construção que definem a estrutura e o conteúdo de uma página web. Elas são usadas para marcar e organizar o texto, criar links, inserir imagens, entre outras funções. Aqui estão algumas das tags mais comuns e suas funções:

Tags Estruturais:

  • < html>: Define o documento HTML.
  • < head>: Contém metadados como título, links para scripts e folhas de estilo.
  • < body>: Contém o conteúdo visível da página.

Tags de Títulos e Texto:

  • < h1> a < h6>: Tags de títulos, com < h1> sendo o mais importante e < h6> o menos.
  • < p>:Define um parágrafo.
  • < br>: Quebra de linha.
  • < strong> e < em>: Destaque de texto em negrito e itálico, respectivamente.

Tags de Links e Imagens:

  • < a>: Cria um link (âncora) para outra página ou seção.
  • < img>: Insere uma imagem na página; usa atributos como src para o caminho da imagem e alt para o texto alternativo.

Tags de Listas:

  • < ul> e < ol>: Criam listas não ordenadas (com marcadores) e ordenadas (com números).
  • < li>: Define um item dentro das listas.

Tags de Formulários:

  • < form>: Define um formulário para entrada de dados do usuário.
  • < input>, < textarea>, < button>: Elementos de entrada de dados, como campos de texto, botões, etc.

Textos

(Voltar)

Os textos no HTML são formatados e estruturados usando diversas tags e atributos que permitem a apresentação de conteúdo de forma organizada, destacada e estilizada. Aqui estão os principais aspectos sobre a formatação de textos em HTML:

Tags de Estruturação de Texto:

  • < p>: Define um parágrafo, criando uma separação clara de blocos de texto.
  • < h1> a < h6>: Definem títulos hierárquicos, onde < h1> é o mais importante e < h6> o menos importante, ajudando na organização e estruturação do conteúdo.
  • < br>: Insere uma quebra de linha dentro do texto, sem iniciar um novo parágrafo.

Tags de Estilização de Texto:

  • < strong>: Aplica negrito ao texto, destacando sua importância sem alterar o estilo visual.
  • < em>: Aplica itálico, indicando ênfase no conteúdo.
  • < b> e < i>: Aplicam negrito e itálico, respectivamente, puramente para efeitos visuais.

Tags de Citação e Código:

  • < blockquote>: Define uma citação em bloco, geralmente usada para destacar citações longas.
  • < q>: Insere citações curtas diretamente no texto, adicionando aspas automaticamente.
  • < code>: Utilizada para exibir trechos de código, geralmente exibidos com uma fonte monoespaçada.

Tags de Formatação Especial:

  • < mark>: Destaca o texto, geralmente com um fundo amarelo, como se tivesse sido marcado com um marcador de texto
  • < del>: Exibe o texto riscado, indicando que algo foi removido.
  • < ins>: Indica uma adição ao texto, geralmente sublinhado.

Listas Ordenadas e Não Ordenadas

(Voltar)

As listas ordenadas e não ordenadas são usadas em HTML para organizar conteúdos de forma clara e estruturada, permitindo criar listas com itens numerados ou com marcadores. Aqui está um resumo sobre cada tipo:

Listas Não Ordenadas (< ul>):

  • Definição: Usam marcadores (bolinhas, quadrados, etc.) para listar itens que não têm uma ordem específica.
  • Uso:Ideal para listas de elementos onde a sequência não importa, como características de produtos, passos não sequenciais, etc.
Estrutura:

< ul>

< li>Item 1< /li>

< li>Item 2< /li>

< li>Item 3< /li>

< /ul>

  • Estilo dos marcadores: Pode ser personalizado com CSS para usar diferentes tipos de marcadores ou até imagens.
  • Listas Ordenadas (< ol>):
  • Definição: Apresentam os itens em uma sequência numerada, indicando uma ordem específica ou hierarquia.
  • Uso: Usada quando a ordem dos itens é importante, como em instruções passo a passo, rankings ou listas de tarefas sequenciais.

Estrutura:

< ol>

< li>Passo 1< /li>

< li>Passo 2< /li>

< li>Passo 3< /li>

< /ol>

  • Tipos de numeração: Pode ser modificada com o atributo type (ex.: números (1), letras (a, A), ou algarismos romanos (i, I)).
  • Listas Aninhadas:É possível criar listas dentro de outras listas, tanto ordenadas quanto não ordenadas, para criar subníveis de itens.

Exemplo:

< ul>

< li>Item Principal

< ol>

< li>Subitem 1< /li>

< li>Subitem 2< /li>

< /ol>

< /li>

< /ul>

Links

(Voltar)

>Os links em HTML são usados para conectar páginas, redirecionar para recursos externos, iniciar downloads, entre outras funcionalidades. Eles são criados com a tag < a> (âncora) e podem ser estilizados de várias formas. Aqui está um resumo sobre os links:

Criação de Links:

  • Tag: A tag < a> é usada para criar links. O principal atributo é href, que especifica o destino do link.

Exemplo Básico:

< a href="https://www.example.com">Visite o site< /a>

  • Atributos Comuns de Links:

href: Define o URL de destino do link. Pode ser um link absoluto (endereço completo) ou relativo (em relação à página atual).

target: Define como o link será aberto. O valor _blank abre o link em uma nova aba ou janela.

< a href="https://www.example.com" target="_blank">Abrir em nova aba< /a>

title: Exibe um texto informativo ao passar o mouse sobre o link, funcionando como uma dica (tooltip).

  • Links de Âncora Interna:

Usados para navegar dentro da mesma página, ligando a um ponto específico definido por um id

Exemplo:

< a href="#section1">Ir para a Seção 1< /a>

< h2 id="section1">Seção 1< /h2>

O link leva diretamente à seção com o id="section1".

  • Links para E-mail:

Abrem o programa de e-mail padrão do usuário para enviar uma mensagem.

Exemplo:

< a href="mailto:example@example.com">Envie um e-mail< /a>

  • Links de Download:

Podem ser configurados para iniciar o download de um arquivo em vez de abrir o link.

Exemplo:

< a href="file.pdf" download>Baixar Arquivo< /a>
